Arreglos:

Homogeneos, seguidos uno de otro. El tamaño es de N<1.

Built in:

   int data[100000];
   data[7]=20;   


Para usar arreglos:

   std::array <int(tipo de dato), N(tamaño arreglo) data>;

La mejor  forma es usar vectores:

   std::vector <int(tipo de dato)>;
   data.resize [100000]


Si no sabemos el tamaño del vector, sea por importar archivos escribimos data.push.back(...) o usar el data.size Así él me define el tamaño
